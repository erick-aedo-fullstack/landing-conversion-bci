Landing de Conversión Crédito Flexible Bci

Landing page enfocada en conversión para el producto Crédito Flexible con respaldo Fogape. Proyecto diseñado y desarrollado desde cero con foco en claridad, jerarquía de contenido y optimización de la experiencia del usuario.



Demo
https://erick-aedo-fullstack.github.io/landing-conversion-bci/



Contexto del proyecto

El objetivo de esta landing es facilitar la comprensión del producto financiero y guiar al usuario hacia la simulación de crédito, reduciendo fricción en la toma de decisión.

Se trata de una solución orientada a usuarios del ecosistema digital bancario que requieren financiamiento claro, accesible y confiable.


Objetivo de negocio

- Aumentar la tasa de simulación de crédito
- Mejorar la comprensión del producto
- Reducir la fricción en el proceso de decisión
- Guiar al usuario hacia una acción concreta (conversión)


Enfoque UX/UI

El diseño fue abordado desde un enfoque centrado en el usuario, priorizando:

- Jerarquía visual clara (lectura rápida y escaneable)
- Reducción de carga cognitiva
- Contenido estructurado en bloques simples
- Llamados a la acción visibles y consistentes
- Diseño responsive (adaptado a distintos dispositivos)


Implementación técnica

- Desarrollo en HTML5, CSS3 y JavaScript
- Estructura optimizada para carga rápida
- Código limpio y enfocado en performance
- Implementación de dataLayer para tracking de eventos


Estrategia de medición

Se considera la integración con herramientas de analítica mediante dataLayer, permitiendo medir eventos clave como:

- Click en CTA principal (simulación de crédito)
- Interacción con secciones relevantes
- Profundidad de scroll
- Comportamiento del usuario dentro de la landing

Esto permite alimentar estrategias de optimización continua (CRO).



Decisiones clave

- Simplificación del contenido para facilitar la comprensión
- Enfoque en una única acción principal (simular crédito)
- Uso de jerarquía visual para guiar el flujo del usuario
- Eliminación de elementos innecesarios que generen distracción


Resultado esperado

Una landing clara, directa y orientada a conversión, capaz de:

- Guiar al usuario sin fricción
- Mejorar la tasa de interacción
- Aportar valor tanto al negocio como a la experiencia del usuario

Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- DataLayer (tracking)
